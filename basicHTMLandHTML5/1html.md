# Elements
- h1, h2, h3, h4, h5, h6 etc. are headings
- h1 for main heading, h2 for subheadings etc.
- p is a paragraph
- div is a division

# Open/Close tags
- All HTML tags are written in lowercase
- <h1>
- </h1>
- <p> "Lorem ipsum" often used as placeholder text
- </p>
- <div>
- </div>

# Comments
- Start with <!-- and end with a -->
