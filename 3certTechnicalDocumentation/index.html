Basic HTML

Basic Elements include: headings(h1, h2, h3 etc.), paragraphs(p) and divisions(div).

All HTML tags are written in lowercase, and must have open tags, e.g. <h1> and closing </h1> with a /.

Basic Attributes

Basic attributes include `src`, and `alt`, for instance, when adding an <img>. For example:

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="relaxing cat">
  • The src attribute is used to specify the URL of the image you want to display.
  • The text inside an alt attribute is used for screen readers to improve accessibility and is displayed if the image fails to load.
Anchor Elements

Anchor elements are used to create hyperlinks that connect to other web pages, files, locations within the same page or any other URL.

The open and close tags are <a> and </a>.

Anchor elements need a destination web address and an anchor text. For example:

<a href="https://www.freecodecamp.org">Click here</a>

The browser will display the text "Click here" as a link.

Links to internal pages

Links to different sections within a webpage are created by assigning a link's href attribute to a # hash symbol followed by the value of the id attribute of the element you want to link to. For example:

<a href="#contacts-header">Contacts</a> where the target destination is:

<h2 id="contacts-header">Contacts</h2>

When users click the Contacts link, they'll be taken to the section of the webpage with the Contacts heading element.

Lists

Unordered lists and bullet points

Holds a collection of list items that are not ordered.

  • List tags are <ul> and </ul>
  • With <li> elements for each bullet point

Ordered lists

  • List tags are <ol> and </ol>
  • With <li> elements for each bullet point
Sections

Sections are used to define sections in a document, such as chapters, headers, footers, or any other sections of the document.

It is a semantic element that helps with SEO and accessibility. For example:

<section>
<h2>Cat Lists</h2>
<h3>Things cats love:</h3>
<ul>
<li>cat nip</li>
<li>laser pointers</li>
<li>lasagna</li>
</ul>

</section>
